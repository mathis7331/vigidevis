# 📁 Dossier Public - Fichiers Statiques

Ce dossier contient les fichiers statiques accessibles publiquement.

## 🎬 Pour la vidéo démo

Placez vos fichiers vidéo ici :

```
public/
  ├── demo-video.mp4      ← Votre vidéo de démonstration
  ├── video-poster.jpg    ← Image de prévisualisation (thumbnail)
  └── README.md           ← Ce fichier
```

## 📹 Formats recommandés

**Vidéo :**
- Format : MP4 (H.264)
- Résolution : 1920x1080 ou 1280x720
- Durée : 30-90 secondes
- Taille : < 20 MB

**Poster :**
- Format : JPG ou PNG
- Dimensions : 1920x1080 px
- Taille : < 500 KB

## 🔗 Accès

Les fichiers sont accessibles via :
- `/demo-video.mp4` dans votre code
- `http://localhost:3000/demo-video.mp4` dans le navigateur

## 📝 Note

Si vous n'avez pas encore de vidéo, vous pouvez :
1. Temporairement commenter la section `<VideoSection />` dans `app/page.tsx`
2. Ou utiliser une vidéo placeholder depuis YouTube/Vimeo (voir VIDEO_SETUP.md)
